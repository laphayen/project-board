# project-board

게시판 기능을 구현한 서비스입니다. 스프링 부트와 관련 기술을 사용해서 만들었습니다.

* * *

### 개발 환경

Intellij IDEA Ultimate 2023.03.06
Java open jdk 22
Gradle 8.7
Spring Boot 3.2.4

* * *

### 기술 스택

* Spring Boot Actuator
* Spring Web
* Spring Data JPA
* Rest Repositories
* Rest Repositories HAL Explorer
* Thymeleaf
* Spring Security
* H2 Database
* MySQL Driver
* Lombok
* Spring Boot DevTools
* Spring Configuration Processor

추가 라이브러리
* QueryDSL 5.0.0
* Bootstrap 5.3.3
* Heroku
